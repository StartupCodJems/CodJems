Sobre o Programa

1. Como funciona o programa de indicações?

Nosso programa recompensa indicações bem-sucedidas. Quando você indica um cliente que contrata nossos serviços, você recebe uma comissão sobre o valor do projeto.

2. Quanto posso ganhar por indicação?

A comissão varia de 5% a 15% do valor do projeto, dependendo do tipo de serviço contratado. Valores exatos são informados no momento da indicação.

Pagamentos

3. Quando recebo minha comissão?

O pagamento é realizado 15 dias após o cliente indicado efetuar o pagamento integral do projeto.

4. Quais são as formas de pagamento?

Pagamos via PIX, transferência bancária ou PayPal. Você escolhe no momento do cadastro.

Processo de Indicação

5. Como faço para indicar alguém?

Basta preencher nosso formulário de indicação com os dados do cliente potencial. Nós entramos em contato com ele e cuidamos de todo o processo.

6. Posso indicar mais de uma pessoa?

Sim! Não há limites para indicações. Quanto mais clientes válidos você indicar, mais você pode ganhar.

Regras e Termos

7. Quais são os critérios para uma indicação válida?

A indicação é válida quando:
1- O cliente não tinha contatado anteriormente conosco;
2- O projeto é efetivamente realizado;
3- O pagamento é concluído.

8. Existe algum custo para participar?

Não! O programa é totalmente gratuito para quem indica. Você só ganha com as indicações bem-sucedidas.
